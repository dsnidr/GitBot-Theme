Introducing GitBot

Automatic GitHub repository status updates in Discord

Add GitBot to your server

Current Features

Repository Subscription
With GitBot, you can subscribe a Discord channel to a GitHub repository that you own.
Automatic Notifications
When a collaborator pushes new commits or opens/closes a pull request, a message gets automatically sent to any Discord channel subscribed to the target repository.

Planned Features

Account Synchronization
In the future, GitBot will allow you to sync your Discord account with your GitBot account.
Account Lookup
Once account synchronization is implemented, you will be able to use GitBot to find a Discord user's GitHub account and vice versa.
